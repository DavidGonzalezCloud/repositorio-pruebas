Un pull request (solicitud de extracción o incorporación) es una solicitud para fusionar cambios de una rama a otra en un proyecto, generalmente con fines de revisión de código. Es una herramienta de colaboración fundamental en el desarrollo de software que permite a los desarrolladores proponer cambios, solicitar la revisión por parte de otros miembros del equipo y facilitar la discusión antes de integrar el código nuevo.
